🔥 Hubungkan Firebase (Real-Time)

Agar semua device sinkron real-time, ikuti langkah berikut:

1
Buka console.firebase.google.com → klik Add project → beri nama bebas
2
Setelah project dibuat, klik ikon </> (Web App) → daftarkan → copy kode firebaseConfig
3
Di sidebar kiri: Build → Realtime Database → Create database → Start in test mode
4
